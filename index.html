Goals Set for a Two Year period

Year 1: Strengthening Software Development Skills and Gaining Cybersecurity Foundations

Q1-Q2: Achieve Senior Software Developer Position

  • Enhance Technical Skills:
    • Imporve mobile development skills with kotlin or flutter
    • Improve backend skills with Node.js and Java Spring boot
    • Optimize performance and scalability of web applications.
    • Improve Animation skills
  • Leadership:
    • Take on more leadership roles in projects.
    • Participate in or lead code reviews.
  • Certifications:
    • Obtain relevant certifications like AWS Certified Developer or Google Professional Cloud Developer.

Q3-Q4: Begin Cybersecurity Education

  • Foundational Knowledge:
    • Enroll in introductory cybersecurity courses (e.g., Coursera, Udemy, or university programs).
    • Study fundamental concepts: network security, encryption, threat modeling, and security best practices.
  • Certifications:
    • Start with basic certifications like CompTIA Security+ or Certified Information Systems Security Professional (CISSP).
  • Practical Experience:
    • Participate in Capture The Flag (CTF) competitions.
    • Start contributing to open-source security projects.
  • Networking:
    • Join cybersecurity forums, attend webinars, and participate in local meetups or conferences.

Year 2: Deepening Cybersecurity Knowledge and Experience

Q1-Q2: Advanced Cybersecurity Learning

  • Specialize:
    • Choose a cybersecurity specialization: penetration testing, threat intelligence, incident response, etc.
    • Enroll in advanced courses relevant to your chosen specialization.
  • Certifications:
    • Obtain advanced certifications: Certified Ethical Hacker (CEH), Offensive Security Certified Professional (OSCP), or Certified Information Security Manager (CISM).
  • Hands-on Practice:
    • Set up a home lab for practicing various cybersecurity techniques.
    • Engage in bug bounty programs.

Q3-Q4: Integrating Cybersecurity with Software Development

  • Secure Development Practices:
    • Implement secure coding practices in your development work.
    • Conduct security audits and vulnerability assessments on your projects.
  • Cross-functional Collaboration:
    • Work with security teams in your organization.
    • Contribute to or lead security-focused projects within your company.
  • Networking and Professional Development:
    • Continue building your professional network in the cybersecurity community.
    • Attend advanced cybersecurity conferences and workshops.
  • Job Transition Preparation:
    • Update your resume to highlight both software development and cybersecurity skills.
    • Start applying for cybersecurity positions or hybrid roles that allow you to leverage both skill sets.
